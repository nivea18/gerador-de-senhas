# Gerador de Senhas

Projeto de uma API Fullstack com React (Vite) e Python (FastAPI)

## Como rodar o Backend
1. Entrem na pasta: `cd backend`
2. Criem o ambiente virtual: `python -m venv venv`
3. Ativem o venv: `venv\Scripts\activate` (Windows)
4. Instalem as dependĂªncias: `pip install -r requirements.txt`
5. Rodem a API: `python main.py`

## Como rodar o Frontend
1. Entrem na pasta: `cd frontend`
2. Instalem as dependĂªncias: `npm install`
3. Rodem o projeto: `npm run dev
